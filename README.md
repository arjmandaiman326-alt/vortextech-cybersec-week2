# VortexTech Cybersecurity Internship – Week 2

## Overview

This repository contains my Week 2 internship task for the VortexTech Cybersecurity Internship Program.

The objective of this task was to gain hands-on experience with two essential cybersecurity concepts:

- Password Strength Evaluation using Python
- Basic Network Port Scanning using Nmap

---

## Project Files

- `password_checker.py` – Python program that evaluates password strength.
- `Week2_Report.pdf` – Detailed report explaining the implementation, scan results, and security analysis.
- `screenshots/` – Screenshots demonstrating the password checker outputs and Nmap scan results.

---

## Password Strength Checker Features

The program evaluates passwords using the following criteria:

- Minimum length (8 characters)
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters
- Detection of common passwords
- Strength rating (Weak, Medium, Strong)
- Suggestions for improving weak passwords

---

## Port Scanning

Nmap was used to perform a safe scan on localhost.

Command used:

```bash
nmap localhost
```

The scan showed that all commonly scanned TCP ports were closed, indicating that no network services were actively listening for incoming connections.
